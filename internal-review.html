Methodology Notes

Analysis Framework: Four-layer diagnosis per GOAL campaign analysis SOP — (1) Client context, (2) Campaign trends, (3) Source-level three-metric funnel (Target Rate → Bid Rate → Win Rate), (4) RightPricing market calibration.

Disposition Correction: The original import treated "Transferred" (83 leads) and "Quoted without contact" (69 leads) as contacted=1. Per GOAL rules, neither is contacted. The disposition CSV file has been fixed (verified 2026-04-17). All contact/quote/sale rates in this report use the corrected mapping. The GOAL dashboard will catch up once the corrected file is uploaded via Conversion Bulk Import → Overwrite.

Target Position: Default target is just above 2nd position RPC (delivers 50-65% of clicks at efficient pricing). Raising bids beyond 1st RPC only justified when Stephen explicitly wants aggressive growth.

Target Multiplier Formula: (Target RPC / Base Bid) × 100%. Target RPC = 2nd RPC × 1.05 (volatility buffer).

Account Maturity: 25 days live. CPA/Sold data is too thin for statistical read (1-2 binds per campaign). Analysis leans on leading indicators: Win Rate, Contact Rate, Quote Rate.
